dnnl_graph_status_t DNNL_GRAPH_API dnnl_graph_set_constant_tensor_cache ( int  flag)

Controls the enabling or disabling of constant tensor cache.

This API must be called once before compilation to take effect.

Parameters
flagSet to positive value to enable the cache and set to 0 to disable the cache. Negative values are invalid.
Returns
dnnl_graph_invalid_arguments if the flag value is invalid, and dnnl_graph_success on success.
