Detailed Description

Engine represents a device (a CPU or a GPU card) and its context.

A graph should be created with a specific engine kind and all partitions returned from the graph will inherit the engine kind. An engine object with the same kind will be used to compile a partition and generate corresponding kernels. The compiled partition will executed on the specific engine.
