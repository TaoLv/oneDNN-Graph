Detailed Description

Partition represents a collection of OPs identified by oneDNN graph implementation as the basic unit for compilation and execution.

enum dnnl_graph_partition_policy_t

Enumerator

Best optimization.

dnnl_graph_partition_policy_fusion 

Have fusion.
