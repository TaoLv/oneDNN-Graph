◆ dnnl_graph_set_constant_cache()

dnnl_graph_result_t DNNL_GRAPH_API dnnl_graph_set_constant_cache ( int  flag)

Control the enabling or disabling of constant cache.

This API must be called once before compilation stage.

Parameters
flagSet to positive value to enable the cache and set to 0 to disable the cache. Negative values are invalid.
Returns
dnnl_graph_result_error_invalid_argument if the flag value is invalid, and dnnl_graph_result_success on success.
